## v5.0.0版本更新内容
- 🔄 **翻译管线全面重构**：引入 Aho-Corasick 自动机、三阶段翻译策略（消歧→翻译→自校验）、格式感知分片与 fallback 回退机制
- ⚡ **翻译性能优化**：文件 I/O O(n²)→O(n)、角色 O(1) 查找、批量上下文扫描，新增 TimingProfiler 性能剖析
- 🌐 **CDN 优选功能**：自动测速并选择最快 CDN 节点（Cloudflare + CloudFront），支持自启配置与缓存有效期
- 🚀 **游戏加速功能**：集成游戏连接加速能力
- 📦 **文件拖入与模组支持**：支持直接拖入文件识别处理，新增格式模组兼容
- 🎮 **启动器优化**：修复 Win11 无窗口启动、速度调节窗口崩溃、快捷键前台进程判断等多项问题
- 🗂️ **架构优化**：统一 GlobalManager 单例模式、单例化日志管理器、GUI 与前端文件拆分
- 🛠️ **新增 OurPlay 神人汉化**：支持 OurPlay 汉化包下载与安装
- 🧠 **LLM 思考模式**：新增是否开启 reasoning/思考模式的选项
- 🔧 **多项修复**：GitHub 下载链接拼接、双重 CFST 测速、免责声明错位、CD 导入、日志冗余等
