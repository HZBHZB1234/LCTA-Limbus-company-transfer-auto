# 关于 LCTA

LCTA (Limbus Company Transfer Auto) 是一款为《边狱公司》(Limbus Company) 打造的全面辅助工具箱。

## 页面介绍

关于页面包含三个信息卡片：

- **项目介绍**：LCTA 项目的完整介绍，与 GitHub README 内容同步。包括功能概述、使用说明和技术信息。
- **更新内容**：当前版本的最新更新日志，展示新增功能、修复的问题和改进内容。每次版本更新后首次启动会自动弹出此内容。
- **引导**：第一次使用的快速入门指引，帮助你了解 LCTA 的基本操作流程。

## 获取帮助

- **帮助中心**：在任意页面点击标题旁的 **?** 按钮或长按键盘 **W** 键 2 秒打开。
- **设置向导**：侧边栏点击"设置向导"，分步完成初始配置。
- **日志文件**：遇到问题时，打开程序目录下的 `logs/` 文件夹获取详细日志。
