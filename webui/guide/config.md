# 配置汉化 API

在使用在线翻译服务之前，需要先配置相应的 API 密钥和参数。

## 配置步骤
1. **选择翻译服务**：从下拉列表中选择要配置的服务（如 DeepL、百度翻译、ChatGPT 等）。
2. **填写 API 参数**：在左侧的配置项中填写配置，留空即为默认，带有星号的为必填参数。
   - 对于 LLM 通用翻译服务，可以选择预设的 LLM 服务（如 OpenAI、Claude 等），基础地址和模型名称会自动填充。
   - LLM 通用翻译服务 的参数 *系统提示词* *返回格式* *最大生成令牌数* 会自动生成，无需填写。
3. **保存配置**：点击“保存配置”按钮，将当前服务的设置存入程序。
4. **测试 API**：填写完成后可点击“测试 api”按钮验证连接是否正常。无论哪一次翻译前，都建议进行api测试

## API申请
绝大部分翻译服务都需要填写密钥。这可能会产生费用。如果不能接受，请看 `下载汉化包` 页面的 *LCTA-AU* 相关章节。  
个人建议使用 LLM 通用翻译服务，因为LCTA对于大语言模型具有特殊优化。在所有的 LLM 中，个人建议使用 Deepseek，其性价比较高，同时作者进行测试时也使用的Deepseek。  
如果不想要付费，可以考虑使用带有免费额度的 LLM或者翻译服务，比如百度翻译等。请自行查阅相关文档。  
如果您只想要进行测试，请使用空翻译器，这将返回原文。  
再次重申，如果您没有特殊需求，只是想要使用自动翻译，使用LCTA-AU可能是一个更好的选择。

## 常见问题解答

### 翻译失败，报错字样中带有 SSL Error
请关闭加速器或代理后重试。
