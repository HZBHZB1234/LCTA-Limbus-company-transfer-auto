# 配置汉化 API

在使用在线翻译服务（如 DeepL、Google Translate、本地 LLM 等）之前，需要先配置相应的 API 密钥和参数。

## 配置步骤
1. **选择翻译服务**：从下拉列表中选择要配置的服务（如 DeepL、百度翻译、ChatGPT 等）。
2. **填写 API 参数**：根据右侧“服务状态”中的说明，填写必填字段（如 API Key、Secret、端点 URL 等）。
   - 对于 LLM 通用翻译服务，可以选择预设的 LLM 服务（如 OpenAI、Claude 等），基础地址和模型名称会自动填充。
3. **保存配置**：点击“保存配置”按钮，将当前服务的设置存入程序。
4. **测试 API**：填写完成后可点击“测试 api”按钮验证连接是否正常，测试结果会显示在进度窗口中。

## 服务状态
- 右侧卡片显示当前选中服务的描述、支持的语言代码、官方控制台链接和文档链接，方便查阅。

## 注意事项
- API 密钥属于敏感信息，程序默认会加密保存（可在设置中关闭加密）。
- 不同服务可能有不同的计费方式和速率限制，请合理使用。
