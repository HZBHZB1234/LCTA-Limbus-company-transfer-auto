# 首页

LCTA 工具箱的主页面，提供整体状态概览和快速操作入口。

## 状态卡片

首页顶部有四个状态卡片，帮助你快速了解工具箱的配置情况：

- **汉化包状态**：显示当前已安装汉化包的数量。如果显示"未安装"，请前往 **汉化包下载** 或 **安装已有汉化** 页面获取汉化包。
- **启动器配置**：显示 Launcher 的更新模式设置状态。如果显示"未配置"，请前往 **Launcher配置** 页面进行设置。
- **LCTA-AU 更新**：显示自动更新功能的开启状态。可在 **设置** 页面中修改。
- **API 配置**：显示已配置的翻译 API 服务数量。如果显示"未配置"，请前往 **配置汉化API** 页面进行设置。

每种状态卡片的颜色代表状态：
- <span style="color: var(--color-success);">**绿色**</span>：已正确配置
- <span style="color: var(--color-warning);">**黄色**</span>：需要关注或配置
- <span style="color: var(--color-text-secondary);">**灰色**</span>：无法检测或状态未知

## 快速操作

首页底部提供四个快捷按钮：

- **检查更新**：手动检测 LCTA 是否有新版本可用。
- **设置向导**：进入分步引导，帮助你快速完成初始配置。推荐首次使用或更新后使用。
- **帮助中心**：打开帮助抽屉，可查看当前页面帮助、完整使用指南或常见问题解答。
- **下载汉化包**：快速跳转到汉化包下载页面。

## 导航提示

- 左侧边栏按功能分组排列，分为**常用工具**、**管理配置**和**系统**三个组。
- 搜索框可以快速筛选导航项，输入关键词即可。
- 每个导航按钮右侧的 **?** 图标可以打开该页面的对应帮助。
- 在任意页面长按键盘 **W** 键 2 秒即可打开当前页面的帮助文档。
