# 汉化包下载

从多个平台下载最新的汉化包，或下载气泡文本模组。

## 下载操作

在每个汉化源的卡片中点击对应的"下载"按钮即可开始下载。下载进度会实时显示。

## OurPlay 下载配置

### 字体处理选项

OurPlay 下载的汉化包携带了两份相同的字体文件，可选择合适的处理方式：

- **保留原字体**：保留两份字体文件。
- **精简字体**：去除冗余的字体文件，节省磁盘空间。
- **使用本地字体缓存**：使用已缓存的字体文件，避免重复下载。缓存路径可在 **设置** 页面配置。

### 启用哈希校验

下载完成后校验文件完整性。如果不完整，程序会提出警告。建议开启。

### 下载源

- **使用 API 获取链接**：通过 Webnote API 获取下载链接。**具有延迟，不建议使用**。关闭后使用直连下载。

## 零协会下载配置

### 压缩格式

选择下载的汉化包压缩格式：

- **ZIP 格式**：兼容性更好，无需额外安装解压软件。
- **7Z 格式**：压缩率更高，文件更小，但需要 7-Zip 支持。

### 文本下载来源

- **从 GitHub 下载**：直接从 GitHub Release 获取，更新最及时。
- **从公益镜像下载 (Beta)**：通过 Webnote API 代理下载，**具有延迟**，适合 GitHub 访问不稳定的情况。

### 使用代理加速下载

通过代理服务器加速 GitHub 请求。**建议开启**，解决国内访问 GitHub 不稳定的问题。

### 使用本地字体缓存

使用已缓存的字体文件而非重新下载。缓存路径可在 **设置** 页面配置。

### 保存原始文件而不打包

下载的汉化包将被解压并保留原始文件结构，而非保存为压缩包格式。此为历史遗留选项，一般无需开启。

## LCTA-AU 下载配置

LCTA-AU 是一个自动机翻汉化源，翻译延迟仅 1-3 小时，且翻译质量高于 OurPlay 等加速器。如果没有特殊需求，**建议使用本汉化源**进行汉化更新。

### 文本下载来源

- **从 GitHub 下载**：直接从 GitHub Release 获取。
- **从公益镜像下载 (Beta)**：通过 Webnote API 代理下载，具有延迟。

### 使用代理加速下载

通过代理服务器加速 GitHub 请求。**建议开启**。

## 气泡文本下载配置

气泡文本是一个基于官方文本接口的语言气泡实现。关于气泡文本的详细信息，请见 [B站视频](https://www.bilibili.com/video/BV1GVpszcEi9)。

> **小贴士**：气泡文本是直接安装在文本包上的，每次切换或更新汉化包后都需要重新安装。如果希望自动化此过程，请参考 **Launcher 配置** 页面的帮助文档。

### 下载有颜色的气泡文本

下载彩色气泡文本，使用效果见[视频](https://www.bilibili.com/video/BV16T1uBBETa)。

### 下载随机加载文本

下载包含随机加载文本的版本，会在战斗加载页面显示。内容为零协会曾经使用的加载文本。

### 立即安装

下载完成后自动安装气泡文本到当前汉化包。**建议开启**，省去手动安装步骤。
