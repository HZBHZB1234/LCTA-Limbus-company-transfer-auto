# 汉化包下载

从多个平台直接下载最新的汉化包，无需手动寻找。

## 下载选项

### OurPlay 汉化包
- **字体处理选项**：可选择保留原字体、精简字体或使用本地字体缓存。
- **哈希校验**：启用后下载完成后会校验文件完整性，确保无误。
- **使用 API 获取链接**：若启用，通过官方 API 获取最新版本信息。

### 零协会汉化包
- **压缩格式**：选择下载的包为 ZIP 或 7Z 格式。
- **下载来源**：可选择从 GitHub 或公益镜像下载（Beta）。
- **使用代理**：加速下载（推荐开启）。
- **使用本地字体缓存**：复用已有的字体文件，减少下载量。
- **保存原始文件**：不打包，直接保存解压后的文件。

### LCTA-AU 汉化包
- 下载来源同样可选 GitHub 或公益镜像，并支持代理加速。

### 气泡文本
- **有颜色的气泡文本**：勾选后下载带颜色版本。
- **随机加载文本**：下载额外的加载界面文本。
- **立即安装**：下载完成后自动安装。

## 下载操作
- 在每个卡片中点击对应的“下载”按钮即可开始。
- 下载进度会显示在模态窗口中，可以最小化或暂停。
