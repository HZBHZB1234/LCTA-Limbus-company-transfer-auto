# 设置向导

设置向导（原"老年人模式"）是一个分步引导工具，帮助你快速完成 LCTA 的初始配置。

## 如何使用

1. 点击侧边栏的 **设置向导** 按钮进入
2. 选择你感兴趣的功能模块（可多选）
3. 按照页面指引逐步配置各项设置
4. 完成后即可开始使用 LCTA

## 在向导中你会配置

- **基础介绍**：了解 LCTA 的基本功能和使用方式
- **启动器**：配置 MOD 支持、气泡文本、文本美化等自动功能
- **翻译**：了解翻译功能的配置方法
- **管理**：汉化包管理和模组管理

## 提示

- 你可以随时通过侧边栏重新进入设置向导
- 向导底部有"重置引导进度"按钮，可以重新开始
- 在任意页面长按 **W** 键 2 秒或点击页面标题旁的 **?** 按钮可以查看该页面的详细帮助
