# 设置向导

设置向导是一个分步引导工具，帮助你快速完成 LCTA 从游戏路径到高级功能的全套初始配置。

## 如何使用

1. 点击侧边栏的 **设置向导** 按钮进入
2. 按照页面提示逐项配置
3. 每个页面配置完成后点击 **继续** 进入下一步
4. 顶部进度条会显示你当前所在的位置
5. 最后一个页面会显示你的所有设置摘要

## 向导步骤说明

| 步骤 | 页面 | 说明 |
|------|------|------|
| 1 | 开始 | 欢迎页面，可以选择开始或继续之前的配置 |
| 2 | 游戏路径 | 设置《边狱公司》的游戏安装目录 |
| 3 | 功能选择 | 选择你想要使用的功能模块（可多选） |
| 4 | 基础设置 | 自动更新、界面主题等基本选项 |
| 5 | 网络更新 | 网络连接和软件更新相关设置 |
| 6 | 启动器 | 启动器核心配置：更新源、MOD、气泡、美化 |
| 7 | 启动器来源 | 启动器各更新源的详细下载选项 |
| 8 | 翻译设置 | 翻译服务、源语言、翻译优化选项 |
| 9 | 下载设置 | 手动下载汉化包时的详细选项 |
| 10 | 气泡文本 | 气泡文本的下载和安装配置 |
| 11 | 模组 | 模组下载、格式和安装说明 |
| 12 | 数据管理 | 文件存储路径、缓存和清理选项 |
| 13 | 完成 | 查看配置摘要和后续操作建议 |

## 你将配置的功能

- **基础介绍**：了解 LCTA 的基本功能、更新机制和帮助系统
- **启动器模式**：自动汉化包更新、MOD 加载、气泡文本、文本美化
- **手动翻译**：翻译 API 配置、专有名词替换、翻译优化
- **数据管理**：汉化包和模组的存储管理、缓存和清理

## 提示与常见问题

### 如何重新进入设置向导？
侧边栏点击「设置向导」即可重新进入。向导会记住你的进度，显示未完成或更新的步骤。

### 如何重置向导进度？
在向导首页底部，点击「重置引导进度」按钮，所有配置将恢复默认，你可以重新开始。

### NEW 标记是什么意思？
当 LCTA 更新后新增了配置项，对应步骤的标题旁会出现 **NEW** 标记，提醒你查看和确认新的设置。

### 为什么有些步骤被跳过了？
向导会根据你在「功能选择」页面中的勾选，自动显示或跳过相关的配置步骤。你可以在设置页面中随时调整这些配置。

### 如何在任意页面获取帮助？
长按 **W** 键两秒或点击页面标题旁的 **?** 按钮，即可查看当前页面的详细使用说明。
