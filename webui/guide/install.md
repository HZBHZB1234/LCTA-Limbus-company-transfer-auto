# 安装已有汉化

安装你已经下载的汉化包文件，或对汉化包进行修改操作（更换字体等）。

## 使用步骤

1. **设置汉化包目录**：在"汉化包目录"输入框中，选择存放汉化包文件的文件夹。留空则使用程序所在目录。
2. **刷新列表**：点击"刷新列表"按钮，程序会扫描目录中的可用汉化包。
3. **选择汉化包**：在列表中点击汉化包右侧的 <i class="fas fa-check"></i> 按钮选中。
4. **安装汉化包**：点击"安装选中汉化包"按钮，在弹出的进度窗口中等待安装完成。

> **提示**：选择项目请点击项目右侧的 ✓ 按钮，被选中的项目会高亮显示。

## 汉化包目录管理

- **浏览**：点击右侧的文件夹图标选择目录。
- **清空**：点击输入框内的 × 按钮清空目录设置，恢复使用程序所在目录。
- 修改目录后列表会自动刷新。

## 删除汉化包

选中汉化包后点击"删除选中汉化包"按钮。此操作会从磁盘上永久删除该汉化包文件，不可撤销，请谨慎操作。

## 字体管理

### 更换字体
为选中的汉化包更换字体文件：

1. 在汉化包列表中选中一个汉化包。
2. 点击"更换字体"按钮。
3. 在弹出的文件选择窗口中选择字体文件（`.ttf` 格式）。
4. 程序将自动替换汉化包内的默认字体并生成新的汉化包文件，保存在当前目录。

### 导出字体
从系统已安装的字体中提取字体文件保存到指定位置：

1. 点击"导出字体"按钮。
2. 在弹出窗口中搜索并选择想要导出的系统字体（支持搜索过滤）。
3. 选择字体后可以预览字体效果。
4. 点击"导出"按钮，选择保存位置即可。

## 常见问题

- **找不到汉化包**：请确认汉化包目录设置正确，文件是 `.zip` 或 `.7z` 格式的汉化包。
- **安装失败**：请检查游戏路径是否已在 **设置** 页面中正确配置。
- **字体更换后显示异常**：请确保选择的字体文件是有效的 `.ttf` 格式，并包含所需的中文字符。
