# 安装已有汉化

如果你已经下载或获取了汉化包（通常是 `.zip` 或 `.7z` 格式），可以通过此功能快速安装到游戏中。

## 操作指南

1. **设置汉化包目录**
   - 在输入框中指定存放汉化包的文件夹路径，或点击“浏览”选择。
   - 留空则使用程序所在目录。

2. **刷新列表**
   - 点击“刷新列表”按钮，程序会自动扫描目录中的可用汉化包并显示在下方。

3. **选择汉化包**
   - 在列表中点击某个汉化包，然后点击“安装选中汉化包”。
   - 安装过程中会弹出进度窗口，完成后即可在游戏中应用。

4. **删除汉化包**
   - 选中列表中的汉化包，点击“删除选中汉化包”可将文件从磁盘移除（不可恢复）。

## 字体管理
- **更换字体**：选中汉化包后，点击此按钮可以从系统字体列表中选择新字体，替换汉化包内的默认字体。
- **导出字体**：从已安装的系统字体中提取字体文件，保存到指定位置。

## 注意事项
- 安装前建议备份游戏原文件（本工具通常会备份，但以防万一）。
- 汉化包格式必须符合程序要求，否则可能安装失败。
