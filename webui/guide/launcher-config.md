# Launcher 配置

如果你希望汉化包能够自动保持最新、气泡文本自动安装、MOD 自动加载，那么你需要配置 LCTA Launcher。Launcher 通过替换 Steam 启动命令来实现自动化。

## 使用步骤

1. **配置各汉化源的下载选项**（零协汉化、OurPlay、LCTA-AU），各选项说明请参考 **汉化包下载** 页面的帮助文档。
2. **设置工作模式**（详见下方说明）。
3. 点击 **保存配置**。
4. 点击 **复制** 按钮复制生成的 Steam 启动命令。
5. 打开 Steam → 游戏库 → 右键 Limbus Company → 属性 → 通用。
6. 在"启动选项"文本框中粘贴复制的命令。
7. 此后通过 Steam 启动游戏时，Launcher 将自动按配置执行汉化更新和维护。

> **注意**：如果你移动了 LCTA 程序的安装位置，需要重新复制并更新 Steam 启动命令。

## 下载配置

### 零协汉化配置
- **压缩格式**：选择 ZIP 或 7Z。ZIP 兼容性更好，7Z 文件更小。
- **文本下载来源**：GitHub 直连或公益镜像（Webnote API，有延迟）。
- **使用代理加速下载**：建议开启，解决国内访问 GitHub 不稳定的问题。
- **使用本地字体缓存**：使用已缓存的字体文件，避免重复下载。

### OurPlay 配置
- **字体处理选项**：保留原字体 / 精简字体 / 使用本地字体缓存。
- **使用 API 获取版本信息**：通过 API 检查更新，建议开启。

### LCTA-AU 汉化配置
- **文本下载来源**：GitHub 直连或公益镜像。
- **使用代理加速下载**：建议开启。

## 工作模式配置

- **更新模式**：选择启动时自动更新的汉化源组合：
  - `不自动更新`：仅启动游戏，不进行任何更新检查。
  - `仅零协更新`：只检查和更新零协会汉化包。
  - `仅ourplay更新`：只检查和更新 OurPlay 汉化包。
  - `仅LCTA-AU更新`：只检查和更新 LCTA-AU 汉化包（推荐）。
  - `更新llc+ourplay`：同时更新零协和 OurPlay。
  - `更新llc+LCTA-AU (github)`：同时更新零协和 LCTA-AU（GitHub 源）。
  - `更新llc+LCTA-AU (API Beta)`：同时更新零协和 LCTA-AU（API 源）。

- **启用 MOD 支持**：启动游戏时加载 Mods 文件夹中的模组。
- **启用文本美化**：更新汉化包后自动应用文本美化规则。相关设置请在 **文本美化** 页面配置。
- **启用气泡文本**：更新汉化包后自动更新气泡文本模组。相关设置请在 **汉化包下载** 页面的气泡文本部分配置。

## Steam 启动命令

Steam 启动命令由程序自动生成，包含了游戏路径和所有配置参数。点击输入框旁的 **复制** 按钮即可复制到剪贴板。
