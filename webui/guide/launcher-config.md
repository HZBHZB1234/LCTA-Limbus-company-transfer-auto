# Launcher 配置

配置游戏启动器的行为，包括自动更新汉化、模组加载等。

## 零协汉化配置
- **压缩格式**：选择下载的零协汉化包格式（ZIP/7Z）。
- **下载来源**：GitHub 或公益镜像。
- **使用代理**：加速下载。
- **使用本地字体缓存**：复用已有字体文件。

## OurPlay 配置
- **字体处理选项**：保留原字体、精简或使用本地缓存。
- **使用 API 获取版本信息**：通过官方 API 获取最新汉化版本。

## LCTA-AU 汉化配置
- 下载来源选择及代理开关。

## 工作模式配置
- **更新模式**：选择自动更新的汉化源组合（如仅零协、仅 OurPlay、同时更新多个等）。
- **启用 MOD 支持**：勾选后启动游戏时会加载 Mods 文件夹中的模组。
- **启用气泡文本**：自动更新气泡文本模组（相关设置在安装汉化包页面）。
- **Steam 命令**：显示用于通过 Steam 启动游戏的命令行，可复制使用。

## 保存
- 完成配置后点击“保存配置”按钮。
- 之后通过本工具启动游戏（或使用复制的 Steam 命令）将按照设定的工作模式运行。
