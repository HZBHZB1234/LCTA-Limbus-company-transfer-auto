# Launcher 配置

我们往往会想，有如此多的翻译源，难以知道他们都是什么时候更新，又或是每次更新文本都需要重新安装气泡文本，过于麻烦。因此，您需要使用LCTA launcher。

下载选项说明这里略过，自行参考 *下载汉化包* 页面。

## 工作模式配置
- **更新模式**：选择自动更新的汉化源组合。每次启动时，将会自动检查对应汉化源的更新，如果有更新，则会立即下载安装。
- **启用 MOD 支持**：勾选后启动游戏时会加载 Mods 文件夹中的模组。
- **启用气泡文本**：自动更新气泡文本模组（相关设置在安装汉化包页面）。

## 保存
- 完成配置后点击“保存配置”按钮。
- 点击复制按钮，复制Steam启动命令
- 打开Steam游戏页面，点击右侧的齿轮，点属性，在新弹出的窗口中将启动命令粘贴进文本框。
- 如果你移动了LCTA启动器，则需要重新进行输入。
