# 已安装数据管理

管理本地已安装的汉化包、模组以及 C 盘数据软链接。
请注意，选择项目需要点击项目右侧的✓按钮。

## 客制化翻译
- **启用客制化翻译**：开关全局汉化功能。关闭后，已安装的汉化包将不会生效。
- **已安装汉化包列表**：显示所有已安装的汉化包，选中后：
  - **使用选中汉化包**：立即切换选中的汉化版本。
  - **删除选中汉化包**：从磁盘移除该汉化包文件夹。

## 模组目录
- **模组安装目录**：可以自定义模组存放路径，默认使用`AppData\Roaming\LimbusCompanyMods`。
- **可用模组列表**：列出所有已安装的模组，每个模组旁边有启用/禁用开关。
- **操作**：刷新列表、打开模组文件夹、删除选中模组。
- 将模组(.carra2或.bank)放入模组文件夹即为安装模组。

## C 盘数据管理
- 管理 `ProjectMoon` 和 `Unity` 数据目录的符号链接，便于将数据迁移到其他盘符。
- 选中文件夹后，可创建、修改或删除软链接。
- 请注意，迁移的目标文件夹应该是一个空文件夹，迁移数据将会把多个文件放入目标文件夹
- 状态标识：
  - **不存在**：目标位置没有该文件夹。
  - **文件夹**：普通文件夹（非链接）。
  - **软链接**：已创建符号链接，点击可打开目标位置。
