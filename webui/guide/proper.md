# 抓取专有词汇

从零协会或其他来源下载专有名词表并解析，生成词汇表供翻译时使用。专有名词表可以帮助翻译服务更准确地翻译人名、地名、技能名等专有术语。

## 使用场景

- 在使用 LLM 翻译前，先抓取专有词汇，提高翻译一致性。
- 生成词汇表用于其他机翻工具的词汇库。
- 更新专有名词表以适配游戏新版本内容。

## 抓取配置

### 输出格式
- **JSON 格式**：生成标准的键值对 JSON 文件。这是 LCTA 翻译功能支持的唯一格式。
- **单文件格式**：所有词汇用分隔符连接成一个文本文件，可用于部分机翻工具的词汇库。
- **双文件格式**：分别保存原词和译词到两个文件，可用于部分机翻工具的词汇库。

### 过滤选项
- **跳过含空格的词汇**：过滤掉包含空格的短语，减少噪音词条。短语类词汇通常不适合作为专有名词使用。
- **最大词汇数量**：限制提取的词条总数，留空表示无限制。建议设置一个合理的上限以避免词汇表过大。
- **最短专有名词长度**：只提取长度大于等于此值的单词，减少单字匹配导致的错误。建议设置为 2 或以上。

### 输出设置
- **分隔符**：仅在单文件格式有效。指定单词之间的分隔符，默认为逗号（`,`）。可以根据目标机翻工具的要求自定义。

## 开始抓取

配置完成后点击"抓取专有词汇"按钮。抓取过程可能需要一定时间，请耐心等待。

## 在翻译中使用

抓取完成后，在 **翻译工具** 页面：
1. 勾选"启用专有名词"
2. 如需使用自定义词汇文件，关闭"自动抓取专有词汇"并手动指定专有名词 JSON 文件路径
