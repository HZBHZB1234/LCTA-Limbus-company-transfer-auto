# 抓取专有词汇

从零协会下载专有名词表并解析，生成词汇表，供翻译时使用。

## 抓取配置
- **输出格式**：
  - JSON 格式：生成标准的键值对文件，翻译仅支持该格式文件。
  - 单文件格式：所有词汇用分隔符连接成一个文本文件，可用于部分机翻词汇库。
  - 双文件格式：分别保存原词和译词，可用于部分机翻词汇库。
- **跳过含空格的词汇**：过滤掉包含空格的短语，减少噪音。
- **最大词汇数量**：限制提取的词条数，留空表示无限制。
- **最短专有名词数长度**：只提取长度大于等于此值的单词，减少错误匹配。
- **分隔符**：仅在单文本格式有效，指定单词之间的分隔符（默认逗号）。
