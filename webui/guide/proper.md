# 抓取专有词汇

从游戏文本中自动提取专有名词（如角色名、技能名、术语等），生成词汇表，供翻译时使用。

## 抓取配置
- **输出格式**：
  - JSON 格式：生成标准的键值对文件。
  - 单文件格式：所有词汇用分隔符连接成一个文本文件。
  - 双文件格式：分别保存原词和译词（通常用于对照）。
- **跳过含空格的词汇**：过滤掉包含空格的短语，减少噪音。
- **最大词汇数量**：限制提取的词条数，留空表示无限制。
- **最短专有名词数长度**：只提取长度大于等于此值的单词，减少错误匹配。
- **分隔符**：仅在单文本格式有效，指定单词之间的分隔符（默认逗号）。

## 操作
1. 根据需要调整上述选项。
2. 点击“抓取专有词汇”按钮。
3. 程序会遍历源语言文本（需在翻译工具的高级选项中预先设置），分析并提取词汇。
4. 完成后会弹出提示，生成的词汇文件保存在指定位置。

## 提示
- 抓取结果可能包含少量非专有词汇，建议手动校对后再用于翻译。
- 此功能依赖源语言文本，请确保在“翻译工具”的高级选项中正确设置了韩/日/英文路径。
