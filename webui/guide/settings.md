# 设置

全局程序设置，影响所有功能的行为。

## 程序设置
- **游戏路径**：指定 LimbusCompany.exe 所在的目录，用于自动定位资源文件。
- **调试模式**：开启后输出更详细的调试日志。
- **启动时自动检查更新**：每次运行自动检测新版本。
- **更新时删除废弃的包**：升级时清理旧版残留文件。
- **更新通过镜像加速**：使用代理/镜像下载更新包。
- **仅更新稳定版本**：只下载正式版，忽略预览版。
- **加密 API 配置**：对保存的 API 密钥进行加密存储。
- **GitHub 代理重试线程数 / 超时秒数**：调整 GitHub 下载的并发和等待时间。
- **启用资源缓存**：将下载的字体等文件缓存到指定目录，减少重复下载。
- **启用数据持久化**：保存一些临时数据到指定路径。

## 配置操作
- **使用默认配置**：将所有选项恢复为程序内置的默认值。
- **重置配置**：清除所有用户配置，还原为出厂状态。
- **检查更新**：手动触发版本检测。
- **强制进行更新**：立即执行更新流程（即使没有新版本）。
- **进入调试页面**：显示隐藏的测试界面（仅供开发者使用）。

## 保存
- 修改设置后记得点击“保存设置”按钮，否则更改可能不会生效。
