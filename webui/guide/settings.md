# 设置

全局程序设置，影响所有功能的行为。修改设置后请务必点击"保存设置"按钮，否则更改不会生效。

## 基本设置

- **游戏路径**：指定 `LimbusCompany.exe` 所在的游戏安装目录。几乎所有功能都依赖此路径，请确保设置正确。
- **调试模式**：开启后显示右键菜单，同时允许进入 Web 开发调试工具（F12）。普通用户建议关闭。
- **启动时自动检查更新**：每次运行程序时自动检测是否有新版本可用。

## 更新设置

- **更新时删除废弃的包**：升级时自动清理新版本不再使用的依赖库文件。建议开启以节省磁盘空间。
- **更新通过镜像加速**：使用 GitHub 代理下载更新包，解决国内访问 GitHub 不稳定的问题。建议开启。
- **仅更新稳定版本**：只下载正式版更新，忽略预览版。一般建议开启（作者基本不发布预览版）。

## 安全与隐私

- **加密 API 配置**：对保存的 API 密钥进行加密存储，防止明文泄露。修改此选项后请在 API 配置页面重新保存。

## 网络设置

- **GitHub 代理重试线程数**：调整从 GitHub 下载时使用的并发线程数。数值越大下载越快，但可能增加请求失败概率。默认 4。
- **GitHub 代理超时秒数**：单个 GitHub 请求的超时等待时间（秒）。网络较差时可以适当调大。默认 8。

## 缓存与存储

- **启用资源缓存**：将下载的字体等资源文件缓存到指定目录，避免重复下载。建议开启。可在下方设置缓存路径。
- **启用数据持久化**：保存临时数据到指定路径，用于存储用户偏好等信息。一般无需修改。

## 配置操作

- **使用默认配置**：将所有选项恢复为程序内置的默认值（保留已保存的 API 配置等关键数据）。
- **重置配置**：完全清空所有配置并重新初始化。**此操作不可逆**，请谨慎使用。
- **检查更新**：手动触发版本检测，查看是否有新版本可用。
- **强制进行更新**：立即执行完整的更新流程，即使当前已是最新版本也会重新下载。
- **进入调试页面**：显示隐藏的测试界面，仅供开发者调试使用。
- **进入已废弃的缓存清理页面**：跳转到旧版缓存清理页面（已停止维护）。

## 保存

修改任何设置后，请务必点击底部的 **保存设置** 按钮。未保存的更改在程序关闭后可能会丢失。
