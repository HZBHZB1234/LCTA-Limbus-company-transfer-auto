## 游戏加速

基于 [OpenSpeedy](https://github.com/game1024/OpenSpeedy) 的游戏速度控制功能，通过 DLL 注入 + Windows 时间 API Hook 实现对 `LimbusCompany.exe` 的变速。

---

### ⚠ 重要风险提示

**DLL 注入技术可能被反作弊系统检测，导致游戏账号封禁。**
请仅在单机/离线游戏中使用，使用者自行承担全部风险。

---

### WebUI 使用

首次进入页面需要同意免责声明才能解锁功能。

1. **进程状态**：页面会自动检测 `LimbusCompany.exe` 的运行状态。
2. **注入**：点击「注入」按钮，将 OpenSpeedy 的 DLL 注入游戏进程。
3. **变速**：通过滑块或预设按钮（1x/2x/3x/5x）选择倍率，点击「应用」生效。
4. **弹出**：点击「弹出」按钮移除 DLL，恢复游戏正常速度。

---

### Launcher 模式

在 Launcher 页面开启「在 Launcher 模式下启用游戏加速」后，通过 Steam 启动游戏将自动注册以下全局热键：

| 热键 | 功能 |
|------|------|
| `Ctrl+S` | 在 1.0x 和设定倍率之间切换 |
| `Ctrl+Shift+S` | 弹出倍率选择窗口，拖动滑块选择临时倍率 |

热键仅在 Launcher 模式下生效，WebUI 页面不注册热键。

### 常见问题

**Q: 杀毒软件报毒怎么办？**
A: DLL 注入行为可能被杀软误报。请将 `openspeedy/data/` 目录添加到杀软白名单。

**Q: 提示"游戏未运行"？**
A: 确保已通过 Steam 启动 `LimbusCompany.exe`，然后刷新状态。

**Q: 提示"权限不足"？**
A: 如果游戏以管理员权限运行，LCTA 也需要以管理员权限运行。

**Q: 速度倍率有什么限制？**
A: OpenSpeedy 支持 0.001x – 1000x，但建议使用 0.1x – 10x 范围。过高倍率可能导致游戏物理引擎错误或崩溃。
