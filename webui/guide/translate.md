# 翻译工具

使用此功能可以将游戏文本（韩文、日文、英文）自动翻译成中文。支持多种翻译服务。

## 主要步骤

1. **配置翻译选项**
   - 启用专有名词：勾选后，翻译时会参考专有名词表（需指定 JSON 文件路径）。
   - 自动抓取专有词汇：若启用，翻译过程中自动提取新词（此时无法使用自定义专有名词文件）。
   - 启用人物角色标识 / 状态效果标识：为 LLM 翻译添加特殊标记，提升上下文准确性。
   - 高级选项：可分别指定韩/日/英文源文本路径及中文输出路径（通常不需要手动设置）。

2. **选择翻译服务**
   - 在下拉列表中选择已配置好的翻译服务（如 DeepL、Google、本地 LLM 等）。
   - 若需要，可勾选“失败时进行相反重试”或“使用文本格式进行请求”（仅 LLM 有效）。

3. **选择源语言**
   - 从下拉框中选择要翻译的源语言：英文、日文或韩文。

4. **开始翻译**
   - 点击“开始翻译”按钮，程序将自动处理文本并生成汉化包。
   - 翻译过程中会弹出进度窗口，可以暂停或取消任务。

## 提示
- 如果未设置文本路径，程序将尝试自动定位游戏目录下的对应语言文件。
- 专有名词文件需为 JSON 格式，包含“原词→译词”的映射。
- “转储过程内容”选项会将中间数据保存到本地，便于调试。
