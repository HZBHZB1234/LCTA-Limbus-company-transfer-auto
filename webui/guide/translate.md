# 翻译工具

使用此功能可以将游戏文本（韩文、日文、英文）自动翻译成中文。支持多种翻译服务。

## 主要步骤

1. **配置翻译选项**
   - 启用专有名词：勾选后，翻译时会参考专有名词表，如果出现大量匹配错误请关闭。这个设置在LCTA-AU中是打开的。
   - 自动抓取专有词汇：若启用，翻译过程中自动重新提取专有词汇，如果想要使用自定义专有名词文件请关闭。
   - 启用人物角色标识 / 状态效果标识：为 LLM 翻译添加特殊标记，提升上下文准确性。
   - 高级选项：可分别指定韩/日/英文源文本路径及汉化原文路径（通常不需要手动设置）如果你没有安装零协会汉化，则需要手动指定汉化原文路径。
   - 转储过程内容: 启用选项后，将会把请求文本与响应文本输出至日志，方便调试。

2. **选择翻译服务**
   - 在下拉列表中选择已配置好的翻译服务（如 DeepL、Google、本地 LLM 等）。
   - 失败时进行相反重试: 如果LLM返回格式错误，则切换请求格式重试。(建议开启，否则会出现不少漏翻)（仅 LLM 有效）
   - 使用文本格式进行请求: 使用自然语言请求返回，且要求使用文本格式返回而不是json格式，建议关闭，否则会出现不少错位。（仅 LLM 有效）。
   - 选择源语言: 从下拉框中选择要翻译的源语言：英文、日文或韩文。LLM翻译无需选择。
